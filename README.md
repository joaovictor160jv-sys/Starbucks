📌 Sobre o projeto

Este projeto é uma landing page inspirada na interface da Starbucks, criada com o objetivo de evoluir minhas habilidades em desenvolvimento front-end.

Além da estrutura com HTML e estilização com CSS, comecei a aplicar JavaScript, dando meus primeiros passos com interatividade.

Um dos diferenciais do projeto é a implementação de um assistente virtual, que adiciona uma camada de interação com o usuário.

💻 Funcionalidades
- Layout moderno inspirado na Starbucks
- Estrutura organizada com HTML
- Estilização com CSS
- Interações básicas com JavaScript
- Assistente virtual integrado
- Manipulação de elementos na página

⚠️ Responsividade

Este projeto foi desenvolvido inicialmente com foco em desktop, sem responsividade.

Ele faz parte do meu processo de evolução e pode receber melhorias futuras para adaptação em dispositivos móveis.
